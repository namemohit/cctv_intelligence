QUICK START (5 MINUTES)
========================

STEP 1: Install
---------------
Windows:
  double-click setup.bat

Linux/Mac:
  chmod +x setup.sh
  ./setup.sh


STEP 2: Configure
-----------------
Edit: config.py

Change this line:
  CAMERA_CONFIG = {
      "entrance": "rtsp://192.168.1.100:554/stream1",
  }

Replace with YOUR camera IP and stream address.


STEP 3: Run Detection
---------------------
Option A - Video file:
  python yolo_dvr_detector.py "C:\path\to\video.mp4"

Option B - Live camera:
  python yolo_dvr_detector.py "rtsp://192.168.1.100:554/stream1"

Option C - Interactive menu:
  python examples.py
  (Select option 1-8)


STEP 4: Check Results
---------------------
Open: detections_log/ folder

Files created:
  • detections_*.csv    (Excel-compatible)
  • detections_*.json   (Structured data)
  • detections_*.db     (Database)
  • yolo_detection_*.log (Debug log)

CAMERA RTSP URLS
=================

Common camera manufacturers:

Hikvision:
  rtsp://username:password@192.168.1.100:554/Streaming/Channels/101

Dahua:
  rtsp://username:password@192.168.1.100:554/stream/main

Uniview:
  rtsp://username:password@192.168.1.100/stream

Generic IP Camera:
  rtsp://192.168.1.100:554/stream1

To find your camera's RTSP URL:
  1. Check camera manual or web interface
  2. Look for "RTSP URL" or "Stream URL"
  3. Format: rtsp://ip:port/stream

TROUBLESHOOTING
================

Q: "Module not found" error
A: Run setup.bat or setup.sh again
   pip install -r requirements.txt

Q: RTSP won't connect
A: Check URL format: rtsp://ip:port/stream
   Test with ffprobe: ffprobe rtsp://...

Q: Slow processing
A: Add --skip-frames 5
   Reduces frames processed for speed

Q: Out of memory
A: Use yolov8n.pt (nano model - already default)
   Set BATCH_SIZE = 1 in config.py

Q: GPU not detected
A: Install CUDA from nvidia.com
   GPU detection is automatic

WHAT GETS LOGGED
=================

For each detection:
  • Exact timestamp
  • Camera/source name
  • Frame number
  • Object class (person, car, etc.)
  • Confidence score (0-1)
  • Bounding box coordinates
  • Object dimensions (width × height)

Example detection log entry:
  2024-03-13 12:30:45 | Camera_1 | person | confidence: 0.92 | Box: [100,50,300,400]
